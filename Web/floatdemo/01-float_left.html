CSS 设计float属性的主要目的，是为了实现文本绕排图片的效果。
为了实现 文本绕排图片 的效果，
HTML文档中要有图片标签 <img> 和容纳文本的标记，我们可以使用<p> ；
同时 <img> 必须出现在 <p> 之前;
     <img src="imges/float-left.jpg" />
     <p>... the paragraph text <p/>
CSS规则如下：
     p { margin:0; border:1px solid red ; }
     img { float:left; margin:4px ; }
以上规则会让图片浮动到左侧，从而让文本绕排到右侧。
