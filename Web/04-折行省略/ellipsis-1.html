css 文本两行显示，超出省略号表示

重点：text-overflow: ellipsis;只对display：inline；起作用
例子： 

            .ellipsis-1 {
                overflow: hidden ;
                text-overflow: ellipsis ;
                white-space: nowrap;           
            }
            span.test{       
                border:1px solid #000000;
            }
<span class="test ellipsis-1" >
This is some long text that will not fit in the box .This is some long text that will not fit in the box .
</span>

效果如下:你会发现text-overflow: ellipsis 失效

This is some long text that will not fit in the box .This is some long text that will not fit in the box .

常见的问题:text-overflow: ellipsis 失效

text-overflow:ellipsis;
only works when the following are true:
(1) The element's width must be constrained in px (pixels). Width in % (percentage) won't work.
(2) The element must have overflow:hidden and white-space:nowrap set. 

        样式表修改为:
        span.test{   
            display:inline-block;
            width:15em;         
            border:1px solid #000000;
        }
